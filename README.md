# Projeto para Estudo

Este projeto tem como objetivo o desenvolvimento de um sistema financeiro pessoal completo, com foco no gerenciamento de finanças e no acompanhamento de uma carteira de investimentos.

O projeto será desenvolvido de forma incremental, ou seja, novas funcionalidades serão planejadas e implementadas conforme sua evolução. Além de servir como uma ferramenta prática, o principal objetivo é aprofundar conhecimentos em desenvolvimento back-end, banco de dados, arquitetura de software e integração entre diferentes tecnologias.

## Tecnologias previstas

* Python
* Flask
* PostgreSQL
* JavaScript

Ao longo do desenvolvimento, outras tecnologias e ferramentas poderão ser incorporadas conforme a necessidade do projeto.

## Objetivos do projeto

* Gerenciamento de receitas e despesas.
* Controle de contas fixas e variáveis.
* Dashboard com indicadores financeiros.
* Acompanhamento de carteira de investimentos.
* Geração de relatórios e gráficos.
* Evolução contínua da arquitetura e das funcionalidades.
* Aplicação de boas práticas de desenvolvimento e organização de código.

> **Observação:** Este é um projeto de estudo e aprendizado contínuo. Seu escopo poderá ser expandido conforme novos conhecimentos forem sendo adquiridos.
